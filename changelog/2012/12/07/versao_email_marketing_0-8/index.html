Versão Email Marketing - 0.8

2012/12/07

Essa versão ficou mais robusta por que nos baseamos em elementos do Email Marketing que é um produto mais completo em relação ao Gateway de Pagamento. Incluídmos novos elementos e melhoramos outros. Como não tínhamos uma política de anotar os changelogs, aqui vai o que lembramos que foi publicado nessa versão.

Iconização com font icons.
Nova forma de iconização utilizando fonts. Inserimos mais de 300 ícones e criamos uma forma alternativa de iconizar os elementos utilizando font-face.
Modernizr 2.6
Atualização da Modernizr 2.6
Customização de Selects
Ninguém gosta de customzar selects, apenas os designers. ;) O problema é que as vezes isso é necessário, logo utilizamos um plugin muito bom que mantém a acessibilidade do usuário e a praticidade de customização via CSS, o plugin se chama Chosen.
Máscaras de formulário
Agora é possível definir máscaras para campos de formulários de uma maneira fácil. Dá uma [lida aqui](http://developer.locaweb.com.br/locawebstyle/formularios/mascaras-formularios/) e veja todas as possibilidades.
Travar letras em inputs
Utilizando a classe **numbersOnly** você consegue travar a inserção de letras em campos de texto, permitindo apenas números.
Esquema de cor VINHO
Novo esquema de cor para contemplar o painel de DNS.
Progress Bar
Um padrão simples de criar [progress bar](/locawebstyle/elementos/barras-de-progresso/).
Balão de informações
Chamamos de [Popover](/locawebstyle/elementos/baloes-de-informacao/). É um balão de informação em elementos que necessitam de explicação ou orientações para o usuário. ;-)
Path ou Passo a Passo
Layout para processos que necessitam de um [passo a passo](/locawebstyle/layout/paths/).
Tradução do calendário
Tradução do calendário utilizado nos sistemas.
Slider
Inserção de exemplo de Slider

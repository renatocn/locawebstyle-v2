Collapse

Agrupando grandes quantidades de informações.

Collapses são usados para organizar grandes quantidades de conteúdo que trazem uma informação ou comportamentos do sistema para a interação com o usuário.

No Locaweb Style alteramos o visual que vem do boostrap, basta inserir a classe ls-collapse. Veja os exemplos abaixo:

Utilização com atributos

Dentro do elemento .panel-heading deve haver qualquer elemento HTML com os seguintes atributos: data-toggle="collapse", que avisa que aquele trigger se refere a um collapse e data-target="#id" que define o collapse alvo.

Atenção:

Se você estiver usando um link como trigger, o atributo data-target é dispensável. Nesse caso você pode colocar o alvo direto no atributo href="#id" do link.

Utilização com Javascript

Ative um collapse manualmente usando o código abaixo:

$(".collapse").collapse();

Métodos

Ativa seu conteúdo como um elemento de collapse.

$('#myCollapsible').collapse({
	toggle: false
});
método descrição
.collapse('toggle') Alterna entre show e hide um elemento de collapse.
.collapse('show') Abre um elemento collapse.
.collapse('hide') Fecha um elemento collapse.

Eventos

Evento Descrição
show.bs.collapse Este evento é disparado imediatamente quando a instância show é chamada.
shown.bs.collapse Este evento é disparado quando um elemento de collapse está visível para o usuário. Irá esperar pela transição do CSS ser completada.
hide.bs.collapse Este evento é disparado imediatamente quando o método hide for chamado.
hidden.bs.collapse Este evento é disparado quando um elemento de collapse ficou escondido para o usuário. Ele espera até as transições do CSS terminarem. 
$('#myCollapsible').on('hidden.bs.collapse', function() {
	// Faça algo aqui
});

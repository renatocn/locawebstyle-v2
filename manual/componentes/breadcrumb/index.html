Breadcrumb

Indica a localização da página atual dentro de uma hierarquia de navegação.

Breadcrumbs são listas ordenadas simples que indicam onde o usuário está juntamente com usa hierarquia de navegação.

O funcionamento é simples. Veja um exemplo abaixo:

  1. Início


    
	
  1. Início
    2.

Exemplo com mais páginas visitadas:

  1. Início
  2. Mensagens
  3. Contato
    
	
  1. Início
    2. 
	
  2. Mensagens
    3. 
	
  3. Contato

Tracking de eventos do Analytics

Registrando eventos de forma automática. Fácil, rápido e efetivo.

O que é?

É uma forma de registrar os eventos comuns em projetos com Locastyle, por exemplo se a ação está abrindo um collapse ou fechando, se é um link, se está abrindo um modal e etc.

Como usar?

Basta ter o objeto ga configurado e o track de eventos do locastyle começa a funcionar.

Requisitos

Este componente funciona apenas na versão V2 do Locastyle e suporta apenas o novo analytics, ou "Universal Analytics".

Se você ainda usa o antigo, procure atualizar, existem documentações de como fazer este upgrade.

Este é um exemplo de disparo de evento no código antigo:

_gaq.push(['_trackEvent', 'Videos', 'Play', 'Baby\'s First Birthday']);

E este é um exemplo de disparo de evento no código novo:

ga('send', 'event', 'category', 'action', 'opt_label', opt_value, {'nonInteraction': 1});

E se eu não quiser usar?

Se você não tiver o código ga configurado o componente não faz nada, mas se você tiver e mesmo assim quiser desativar o track de eventos, pode usar a forma padrão de desabilitar os eventos.

window['ga-disable-UA-XXXXXX-Y'] = true;

Veja a documnetação de como desabilitar o tracking de eventos.

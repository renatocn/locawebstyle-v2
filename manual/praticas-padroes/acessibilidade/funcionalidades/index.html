Funcionalidades Gerais

Melhorias acessíveis no Locaweb Style

Dentro do nosso framework Locaweb Style foram inseridas funcionalidades para a acessibilidade, conheça elas:

Submenu Acessível

Caso o projeto possua submenu, já o deixamos acessível via teclado. Isso ajuda muito quem utiliza leitores de tela.

Dando foco ao input

Em alguns casos precisaremos dar foco ao input dentro de modal ou collapse, basta inserir a classe auto-focus e a mágica acontece, pode parecer algo simples, no entanto essa funcionalidade ajuda muito o usuário.

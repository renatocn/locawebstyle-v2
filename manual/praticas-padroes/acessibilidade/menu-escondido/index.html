Menu escondido

Menu acessível apenas para leitores de tela e escondido visualmente.

Para facilitar a navegação via tabs, é comum criarmos um menu escondido para que os leitores de tela o leiam antes de todas as outras opções da página. Esse menu contém sempre atalhos priorizados para que as tarefas do serviço sejam executadas mais rapidamente. Você pode montar um menu escondido de maneira muito simples no Locaweb Style.

As informações que estarão neste menu oculto será definida inserindo o atributo data-access="true". Adicionando esse atributo em algum link, botão e etc, automaticamente nosso script irá adicionar a ação ao menu. Veja um exemplo:

Sugerimos que o menu não tenha mais de 3 ítens. É muito importante que ele seja curto para que o usuário não se perca... caso contrário o menu perde todo o sentido. Faça um estudo em seu projeto para entender quais ações em cada página é mais importante e insira um atalho no menu escondido.

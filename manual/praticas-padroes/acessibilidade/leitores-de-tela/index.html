Leitores de tela

Conheça quais são e como funcionam.

Hoje em dia existem excelentes leitores de tela que podem atender as mais diversas necessidades dos deficientes visuais, alguns são pagos e outros gratuitos.

Listamos aqui os leitores de tela mais utilizados.

NVDA

O NVDA é um leitor de telas excelente, e, além disso, gratuito, e que pode rodar nos sistemas windows 2000, XP, vista e windows7. Com tecnologia open source, pode ser instalado livremente em qualquer computador e mesmo modificado para se adaptar a determinadas situações, ou para receber melhorias.

Baixar NVDA

Atalhos do NVDA

  1. INSERT + numpad 5: fala o objeto atual;
  2. INSERT + numpad8: move para o objeto pai (indo na direção da raiz da árvore);
  3. INSERT + numpad4: move para o objeto anterior (o objeto diretamente anterior a este, no mesmo nível);
  4. INSERT + numpad6: move para o próximo objeto (o objeto diretamente posterior a este, no mesmo nível);
  5. INSERT + numpad2: move para o primeiro objeto filho (o primeiro galho partindo deste objeto);
  6. INSERT + shift + numpad4: move para o objeto anterior no curso (automaticamente cruza a fronteira dos objetos pai e filho até que possa ir "voltando");
  7. INSERT + shift + numpad6: move para o próximo objeto no curso (automaticamente cruza a fronteira dos objetos pai e filho até que possa ir adiante);
  8. INSERT + numpadMenos: move para o foco;
  9. INSERT + numpadEnter: ativo o objeto atual (isto é, como pressionar enter / clique / duplo clique);
  10. INSERT + numpadDivisão: move o mouse parao objeto atual;
  11. INSERT + numpadMultiplicação: move para o mouse;
  12. INSERT + shift + numpad5: anuncia as dimensões do atual objeto em função da tela;
  13. INSERT + numpadMais: fala todos os objetos - começando a partir o objeto atual e seguindoem curso.

Nota: As teclas numpad são as que estão localizadas no teclado numérico. Para que as teclas numpad funcionem corretamente, é necessário que a tecla numlock esteja desligada.
INSERT pode ser outra tecla, caso o usuário tenha alterado a tecla modificadora.

DOSVOX

O sistema dosvox não é propriamente um “leitor de telas”, e sim um sistema de acesso ao computador para deficientes visuais. Roda sob qualquer versão do sistema windows, além de ser capaz de rodar sob alguns emuladores em outros sistemas, como o wine, no linux.

Tem uma gama variada de programas para atender as mais diversas necessidades, como por exemplo:

  • Editor de textos
  • Calculadora
  • Agenda de compromissos
  • Caderno de telefones
  • Programas específicos para internet, como por exemplo:
  • Correio eletrônico
  • Navegador web
  • Programa de acesso ao twitter
  • Programa para acesso a chat
  • Jogos diversos, em diversas modalidades como:
  • Educativos
  • RPG
  • Oráculos
  • Desafios
  • Dicionários de diversos idiomas
  • Exibidor de apresentações interativas
  • E muito mais!

Contudo, o dosvox proporciona acessibilidade dentro de seu próprio sistema, mas seus recursos para acessibilidade fora do mesmo são muito limitados. Mas o dosvox em conjunto com o NVDA já fazem uma “boa parceria”, e o usuário não precisa gastar absolutamente nada para obtê-los, basta simplesmente fazer download dos programas.

Baixar dosvox

Virtual Vision

O Virtual Vision é um leitor de telas completo e que atende grande parte das necessidades dos deficientes visuais. Surgido em 1997, foi o primeiro software totalmente brasileiro para leitura de telas e é capaz de dar acesso com desenvoltura, por exemplo, a aplicativos como word, excel, outlook e windows live mail, msn, internet explorer e outros.

O virtual vision pode ser obtido gratuitamente através do banco Bradesco, se o usuário tiver uma conta no referido banco, ou, mediante pagamento, através do site da micropower.

Baixar Virtual Vision

JAWS

O jaws é, sem dúvida, um dos mais poderosos leitores de tela para cegos. É capaz de dar uma acessibilidade excelente, inclusive, às tarefas mais avançadas que o usuário possa necessitar executar. Tendo um custo de licença é relativamente alto, contudo continua sendo a melhor opção para quem pode adquirir.

Dá pra utilizá-lo sem possuir licença durante 40 minutos, depois é necessário reiniciar o computador.

Baixar Jaws

VOICEOVER

O VoiceOver não só fala o que está acontecendo, ele ajuda a fazer as coisas acontecerem. Ele fala o que está na tela e ainda orienta você em ações como selecionar uma opção de um menu ou ativar um botão usando seu teclado ou trackpad. Com o VoiceOver, você tem controle total do seu Mac, sem necessidade de ver a tela.

O VoiceOver possue uma voz humana e em português chamada Raquel, já que as vozes de leitores de tela são robóticas.

Atalhos do VoiceOver

  1. Control + option + u = Ativa a navegação via Landmarks, links, titles
  2. Control + option + h = Ativa help com todos os comandos
  3. Control + option + i = Seleciona os itens da página
  4. Control + option + k = Iniciando ajuda do teclado
  5. Control + option + b = Para saber qual é o title da página

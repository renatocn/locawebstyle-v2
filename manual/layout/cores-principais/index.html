Tema de cores

Exemplo de temas de cores usadas no projetos.

Você pode querer mudar a cor principal do seu sistema. Preparamos algumas classes que mudam as cores principais do Locaweb Style e de seus elementos, sem dar tanto trabalho para nós. Essas cores são atualizadas de acordo com os projetos que temos aqui.

Se você quiser mudar as cores de todo o sistema, basta inserir a classe desejada na tag HTML. Se você quiser que os uma parte do layout fique com outras cores, basta adicionar a classe de cor no elemento pai. Veja abaixo a listagem das classes:

.color-blue .color-green-light .color-green

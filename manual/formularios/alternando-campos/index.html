Formulários

Alternando campos de formulário.

Para ativar essa funcionalidade o elemento precisa possuir a classe toggle-pass e receber o atributo data-target com o mesmo valor do ID do input que será alternado seu tipo de password/text.

Funcionamento apenas em formulários

É importante lembrar que o alternador de campos funciona apenas dentro de um formulário, então ele deve estar sempre dentro de um form.
